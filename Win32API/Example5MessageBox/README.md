# Message Box

A C# console application that displays a Windows message box using the Win32 API through P/Invoke.

## Technologies

- C#
- .NET
- Win32 API
- P/Invoke

## Windows API Used

This project uses `MessageBox()` from `user32.dll` to display a native Windows message box.

## Features

- Displays a native Windows message box
- Accepts custom title and message text
- Supports different button and icon styles
