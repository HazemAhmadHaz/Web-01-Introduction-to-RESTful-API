# Shutdown Windows

A C# console application that shuts down, restarts, or logs off Windows using the Win32 API through P/Invoke.
**Note: Running this example will shut down Windows, so make sure to save any work related to your programs before running it to avoid losing anything.

## Technologies

- C#
- .NET
- Win32 API
- P/Invoke

## Windows API Used

This project uses `ExitWindowsEx()` from `user32.dll` to perform the shutdown, restart, or logoff operation.

## Features

- Shuts down the computer
- Restarts the computer
- Logs off the current user
- Displays success or error messages
