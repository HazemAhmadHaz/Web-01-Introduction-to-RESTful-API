# Outlook Email Sender

A C# console application that sends emails through Outlook using COM Interop.

## Technologies

- C#
- .NET
- COM Interop
- Microsoft Outlook Object Library

## API Used

This project uses `Microsoft.Office.Interop.Outlook` to automate Outlook and send emails programmatically.

## Features

- Creates a new mail item
- Sets subject, recipient, and body text
- Sets email importance/priority
- Sends the email through the local Outlook installation

## Preview

No preview available — this project requires a local Microsoft Outlook installation to run.
