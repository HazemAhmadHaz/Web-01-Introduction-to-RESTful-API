# Create An Excel File

A C# console application that creates and saves an Excel file using COM Interop.

## Technologies

- C#
- .NET
- COM Interop
- Microsoft Excel Object Library

## API Used

This project uses `Microsoft.Office.Interop.Excel` to automate Excel and create a spreadsheet programmatically.

## Features

- Launches Excel in the background (invisible)
- Creates a new workbook
- Writes data into cells
- Saves the workbook to a specified file path
- Closes the workbook and quits Excel automatically

## Preview

No preview available — this project requires a local Microsoft Excel installation to run.
