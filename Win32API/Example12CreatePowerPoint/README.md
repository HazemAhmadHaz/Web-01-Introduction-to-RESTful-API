# Create PowerPoint

A C# console application that creates and saves a PowerPoint presentation using COM Interop.

## Technologies

- C#
- .NET
- COM Interop
- Microsoft PowerPoint Object Library

## API Used

This project uses `Microsoft.Office.Interop.PowerPoint` to automate PowerPoint and create a presentation programmatically.

## Features

- Launches PowerPoint in the background
- Creates a new presentation
- Adds a slide with custom text
- Saves the presentation to a specified file path
- Closes the presentation and quits PowerPoint automatically

## Preview

No preview available — this project requires a local Microsoft PowerPoint installation to run.
