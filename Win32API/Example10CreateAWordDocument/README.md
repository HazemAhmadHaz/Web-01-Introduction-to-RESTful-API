# Create A Word Document

A C# console application that creates and saves a Word document using COM Interop.

## Technologies

- C#
- .NET
- COM Interop
- Microsoft Word Object Library

## API Used

This project uses `Microsoft.Office.Interop.Word` to automate Word and create a document programmatically.

## Features

- Launches Word in the background (invisible)
- Creates a new document
- Adds a paragraph with custom text
- Saves the document to a specified file path
- Closes the document and quits Word automatically

## Preview

No preview available — this project requires a local Microsoft Word installation to run.
