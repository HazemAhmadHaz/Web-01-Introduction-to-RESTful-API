# Introduction to RESTful API

This repository contains the examples and projects I completed while studying **Introduction to RESTful API**.

The goal of this repository is to understand how software systems communicate using APIs, starting from Windows APIs and moving toward Web APIs and RESTful APIs.

## Contents

## 🖥️ Windows API

Examples using **Win32 API** through C# and P/Invoke:

- Change Desktop Wallpaper
- Get Screen Resolution
- Battery Information
- Shutdown Windows
- MessageBox
- Process Management
- Audio API
- WiFi Networks
- Send Email via Outlook
- Create Word Documents
- Create Excel Files
- Create PowerPoint Presentations

## 🌐 Web API & RESTful API

Topics and projects covering:

- What is an API?
- XML and JSON
- Web API Fundamentals
- RESTful API Principles
- HTTP Requests and Responses
- HTTP Methods (GET, POST, PUT, DELETE)
- HTTP Status Codes
- Client-Server Communication
- DTO (Data Transfer Object)
- 3-Tier Architecture
- Database Integration
- Image Upload and Retrieval
